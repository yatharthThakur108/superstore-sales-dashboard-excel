# superstore-sales-dashboard-excel
Developed an interactive Super Store Sales Dashboard in Excel to analyze sales, transactions, quantity orders, customer segments, and regional performance. Used charts, KPIs, slicers, and filters to visualize monthly trends and business insights for better decision-making.
# Super Store Sales Dashboard using Excel

## Project Overview
This project is an interactive Excel dashboard created using Super Store sales data. The dashboard helps analyze sales performance, transactions, quantity orders, customer segments, and regional insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- KPI Cards

## Features
- Region-wise sales analysis
- Monthly order trends
- Category-wise transaction analysis
- Customer segment insights
- Interactive dashboard filters
